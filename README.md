## Requirements
* [PostgreSQL (9.5+)](https://www.postgresql.org/download/)

## Setup
```sh
git clone https://github.com/ericljiang/settlers.git
cd settlers
./gradlew createdb
./gradlew run
```
Connect to http://localhost:4567
